MI25 Path Worklog

MI25経路観測の作業ログ要約ページです。

いま分かっていること（このワークスペース）

  • 観測・推論・未確定を分離する運用を採用。

言えること（条件つき）

  • run 単位で、観測・推論・未確定を分けて記録するルールを維持しています。
  • phase-window / split など観測手法の差も、同じログ上で比較できる形で残しています。

まだ言えないこと

  • worklog 要約だけで性能因果を確定しない方針です。
  • raw ログ非公開分があるため、最終評価は対応する参照ノートと併読が前提です。
