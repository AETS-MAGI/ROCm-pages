MI25 Index

MI25（GCN5）の調査ページ入口です。観測→比較→記録の手順を基準に整理します。

いま分かっていること（このワークスペース）

  • anchor条件で fallback/dispatch 観測を確認。
  • lane比較で同一runの tok/s 比が最大約9.4xのケースを観測。
  • 観測事実と未確定事項（kernel-level因果）を分離して扱う。

言えること（条件つき）

  • anchor 条件で fallback_confirmed / dispatch_confirmed を再現した、という観測事実は保持できます。
  • 同一 run 比較で最大約 9.4x の tok/s 改善ケースを観測していますが、条件限定の結果です。

まだ言えないこと

  • catalog-read と個別 kernel dispatch の 1:1 因果は、まだ断定できません。
  • 他モデル・他世代へそのまま一般化できるとは言えません。
