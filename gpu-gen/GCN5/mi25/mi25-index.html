MI25 Index

MI25（GCN5）の調査ページ入口です。観測→比較→記録の手順を基準に整理します。

いま分かっていること（このワークスペース）

  • anchor条件で fallback/dispatch 観測を確認。
  • lane比較で同一runの tok/s 比が最大約9.4xのケースを観測。
  • 観測事実と未確定事項（kernel-level因果）を分離して扱う。

言えること（条件つき）

  • ここに記載する結論は、ページ内で明示した条件・観測範囲に限定されます。
  • 再現条件が変わる場合、同じ結果になるとは限りません。

まだ言えないこと

  • kernel / solver レベルの厳密な因果を、ページ単独では断定しません。
  • 未観測のケースや他ワークロードへの一般化は保留します。
