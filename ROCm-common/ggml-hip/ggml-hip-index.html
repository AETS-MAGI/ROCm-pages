ggml-hip

ggml-hip（Ollama の HIP バックエンド）の経路確認状況を記録します。

Records ggml-hip (Ollama's HIP backend) path confirmation status.

観測済み事実Observed Facts

  • MI25 では libggml-hip.so 読み込み確認が重要な証跡の 1 つ。
  • RX9070XT pilot でも HIP 経路観測を先に固定している。
  • MI25 / RX9070XT の両方で、HIP backend 読み込み確認は経路観測の基本証跡として有効。
  • For MI25, confirming libggml-hip.so loading is one of the key evidence points.
  • In the RX9070XT pilot, HIP path observation is fixed first.
  • For both MI25 and RX9070XT, HIP backend load confirmation is valid as a basic path observation evidence.

未確定事項Open Questions

  • backend 読み込み確認のみで性能要因を確定しません。
  • GPU 世代差の要因を ggml-hip 単独で説明し切ることは保留します。
  • Backend load confirmation alone is not used to confirm performance factors.
  • Explaining GPU generation differences solely from ggml-hip is withheld.
