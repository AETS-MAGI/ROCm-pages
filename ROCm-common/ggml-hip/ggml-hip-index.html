The HIP backend for ggml. Distributed as libggml-hip.so loaded by Ollama at inference time. Embeds custom kernels for quantized matmul, Flash Attention, RoPE, and more, targeting multiple AMD GPU architectures including gfx1201.

ggml-hip は ggml の backend abstraction 上に実装された HIP 特化バックエンドです。CUDA 側の ggml-cuda と同一ソース系統（ggml-cuda.cu / mmq.cu / mmvq.cu など）から HIP ビルドされます。vendors/hip.h が cublasCreate 等の CUDA API を hipBLAS 等価物にリマップし、コードの共通化を実現しています。

ggml-hip is the HIP-specialized backend implemented on top of ggml's backend abstraction. It shares the same source family as ggml-cuda (ggml-cuda.cu, mmq.cu, mmvq.cu, etc.), built with HIP. vendors/hip.h remaps cublasCreate and related CUDA APIs to hipBLAS equivalents, enabling a shared codebase across both platforms.
