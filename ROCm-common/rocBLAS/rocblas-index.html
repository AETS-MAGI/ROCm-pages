観測済み事実Observed Facts

  • AETS lane / system lane の観測差が throughput 差として現れるケースを確認。
  • kernel レベルの因果は未確定として扱う。
  • Controlled experiments using ROCBLAS_TENSILE_LIBPATH difference conducted via lane comparison on MI25.
  • Cases confirmed where observation differences between AETS and system lanes appear as throughput differences.
  • Kernel-level causality is treated as undetermined.

現時点での知見Current Findings

  • MI25 では、lane 比較により rocBLAS/Tensile 側の観測差と実行差を条件つきで確認しています。
  • direct 可視性はワークロード依存であるため、単一モデルの結果で一般化しません。
  • For MI25, rocBLAS/Tensile observation and execution differences are conditionally confirmed via lane comparison.
  • Direct visibility is workload-dependent; results from a single model are not generalized.
