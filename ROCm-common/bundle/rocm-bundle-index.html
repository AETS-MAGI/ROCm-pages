Ollama が使用する ROCm ランタイムバンドルの構造と、RX9070XT 実機で確認した配置を記録します。バンドル優先ロードにより、system ROCm とは独立した stack で動作します。

Documents the structure of the ROCm runtime bundle used by Ollama and the layout confirmed on the RX9070XT live machine. Bundle-first loading ensures operation against an independent stack from system ROCm.
