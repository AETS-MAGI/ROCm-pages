hipBLASLt

カスタムデータレイアウト・混合精度・活性化関数融合（GELU/ReLU/Swish）をサポートする柔軟な行列積ライブラリです。TensileLite をバックエンドに持ち、rocBLAS とは独立した GEMM API（hipblasLtMatmul）を提供します。gfx1201 向け sparse GEMM テスト構成を含む 250 超の参照が確認されています。

A flexible matrix multiplication library supporting custom data layouts, mixed precision, and fused activations (GELU/ReLU/Swish). Uses TensileLite as its backend and provides an independent GEMM API (hipblasLtMatmul) from rocBLAS. Over 250 gfx1201 references found in its TensileLite subdirectory.

ROCm スタックにおける位置づけPosition in the ROCm Stack

hipBLASLt は rocBLAS と並列に位置する特化 GEMM ライブラリです。backend に TensileLite（Tensile の軽量版）を持ち、activation fusion や sparse GEMM などの拡張機能を提供します。ggml-hip の現行ソースでは直接の hipblasLt* 呼び出しサイトは主たる dispatch 経路に現れていませんが、rocBLAS 内部ポリシー経由で間接的に呼ばれる可能性は否定されていません。

hipBLASLt is a specialized GEMM library positioned alongside rocBLAS. Its backend is TensileLite (a lightweight Tensile variant), providing extended capabilities like activation fusion and sparse GEMM. In the current ggml-hip source, direct hipblasLt* call sites do not appear in the primary dispatch path, but indirect calls via internal rocBLAS policy cannot be ruled out.
