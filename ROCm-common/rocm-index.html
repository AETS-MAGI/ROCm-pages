ROCm Common Components

世代横断で使う ROCm 関連コンポーネント（数学ライブラリや推論基盤）を整理するページです。

いま分かっていること（このワークスペース）

  • この配下に rocBLAS / Tensile / MIOpen / ggml-hip などの個別ページを配置。
  • MI25調査では、LLM推論の主経路候補を rocBLAS/Tensile 側で重点観測している。
  • コンポーネントごとの最終結論はまだ固定せず、観測ログに沿って更新する方針。

言えること（条件つき）

  • ここに記載する結論は、ページ内で明示した条件・観測範囲に限定されます。
  • 再現条件が変わる場合、同じ結果になるとは限りません。

まだ言えないこと

  • kernel / solver レベルの厳密な因果を、ページ単独では断定しません。
  • 未観測のケースや他ワークロードへの一般化は保留します。
